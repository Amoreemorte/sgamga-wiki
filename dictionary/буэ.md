# Слово "БУЭ"
	бу-э́

# Определение
**Буэ** - это междометная лексическая единица, 
используемая в разговорной речи. 
Основные контексты ее использования:
1. *Капитуляция оппонента во время спора* - после исчерпания аргументов оного он может использовать его в качестве сигнала о прекращении спора,
невербально давая оппоненту понять, что спор далее не может быть продолжен. 
2. *Эмоциональная разрядка* - абсурдная единица, намеренно вводимая для снижения напряженности, а также используемая для создания комического эффекта. 

# Лингвистическая характеристика
**Часть речи**: междометие
**Падежи**: не склоняемое 
**Произношение**: ˈbuɛ
**Синонимы**: отсутствуют
**Антонимы**: отсутствуют
**Принадлежность к языку**: русский язык 
**Примеры применения**:
	
  *Не знаю что ещё сказать. Буэ!* - оппонент исчерпал свои доводы и для невербального прекращения спора завершает свою речь абсурдной вставкой.
  
  *Не беспокойся по этому поводу. Буэ!* - намеренно используется для смещения фокуса внимания с тяжелой ситуации на абсурдное слово. 

*Статус: гипотетическая лексическая единица. Представлено для исследовательских целей.*
